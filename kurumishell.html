快速开始

简单几步，快速部署和使用 KurumiShell

后端启动

1. 克隆代码

git clone https://github.com/MidasLee/KurumiShell.git

2. 进入项目目录

cd KurumiShell

3. 启动后端服务

./gradlew bootRun
确保已安装 JDK 17+ 和 Gradle 7+

前端启动

1. 进入前端目录

cd web

2. 安装依赖

npm install

3. 启动开发服务器

npm run dev
访问: http://localhost:5666

爬虫启动

1. 进入爬虫目录

cd spider/dudubird-spider

2. 安装依赖

pip install -r requirements.txt

3. 启动爬虫服务

python main.py
确保已安装 Python 3.8+

灵活的配置选项，满足不同环境的部署需求

前端环境配置

配置文件：web/.env.development

# 应用标题
VITE_APP_TITLE = "KurumiShell"

# 前端服务端口
VITE_APP_PORT = 5666

# API基础URL
VITE_APP_BASE_URL = "http://localhost:8888"

# 渡渡鸟镜像仓库API地址
VITE_APP_DUDUBIRD_URL = "http://localhost:8166"

# 轩辕镜像仓库API地址
